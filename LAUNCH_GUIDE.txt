## STEP 2 — GET HOSTING (Railway.app — FREE to start)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1. Go to:  https://railway.app
2. Sign up with GitHub
3. Click "New Project" → "Deploy from GitHub repo"
4. Upload your files to a GitHub repo first:
   - Create account at https://github.com
   - Create new repo named "almaroof"
   - Upload: server.js, package.json, almaroof.sql, public/index.html

5. In Railway, add a MySQL database:
   - Click "+ New" → "Database" → "MySQL"
   - Copy the DATABASE_URL it gives you

6. Set environment variables in Railway:
   - Click your project → "Variables" tab
   - Add all variables from .env.example

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
## STEP 3 — SETUP DATABASE
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
In Railway MySQL:
1. Click your MySQL service → "Connect" tab
2. Use the MySQL CLI connection string shown
3. Run: mysql -h HOST -u USER -p DATABASE < almaroof.sql
   (Or use a GUI like TablePlus or DBeaver)

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
## STEP 4 — REGISTER PAYSTACK (Payment Gateway)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1. Go to: https://paystack.com
2. Click "Create free account"
3. Fill in your business details (Al Maroof)
4. Go to Settings → API Keys & Webhooks
5. Copy your:
   - Public Key  → PAYSTACK_PUBLIC_KEY in .env
   - Secret Key  → PAYSTACK_SECRET_KEY in .env
6. Set Webhook URL to:
   https://yourdomain.com/api/paystack/webhook

IMPORTANT — To receive real money:
- Submit your CAC documents to Paystack
- Or start with test mode first

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
## STEP 5 — REGISTER VTPASS (Data/Airtime/Bills)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1. Go to: https://vtpass.com
2. Click "Register" → fill business details
3. Login → Go to Settings → API
4. Copy your:
   - Username/Email → VTPASS_USER in .env
   - Password       → VTPASS_PASS in .env
5. Fund your VTPass wallet (minimum ₦5,000 to start)
6. Test in sandbox first: set VTPASS_ENV=sandbox

VTPass ServiceIDs (already configured in server.js):

- Airtel Data: airtel-data
- Glo Data:    glo-data
- 9mobile:     etisalat-data
- EKEDC:       eko-electric
- IKEDC:       ikeja-electric
- DSTV:        dstv
- GOtv:        gotv

In app.html, find this line near the top of the <script>:
  BASE_URL: 'https://your-api-domain.com/api'

Change it to your actual Railway URL, e.g.:
  BASE_URL: 'https://almaroof-production.up.railway.app/api'

Also add your Paystack public key:
  window.PAYSTACK_PUBLIC_KEY = 'pk_live_xxxxxxxxxxxx';

Place app.html in the "public" folder and rename to index.html
